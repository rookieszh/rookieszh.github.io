DockerFile

Basic Component
  • Docker

Dockerfile的本质就是由一系列顺序编排的命令和参数组成的脚本，并利用它把我们需要的应用制成一个镜像文件

特点

1、对于开发人员：可以为开发团队提供完全一致的开发环境；
2、对于测试人员：可以直接用开发所制成的镜像或者通过Dockerfile文件构建一个新的镜像开始工作了；
3、对于运维人员：在部署时，可以实现应用的无缝移植。
